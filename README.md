🚀 ValidEats — AI Powered Food Health Analyzer

ValidEats is a full-stack AI-powered web application that analyzes packaged food products using OCR + AI to provide health ratings, ingredient insights, and personalized recommendations.

🌐 Live Demo
🔥 Frontend (User Interface):
👉 [https://valid-eats.vercel.app](https://valideats.vercel.app)
⚡ Backend (API Server):
👉 [https://valideats.onrender.com](https://valideats.onrender.com)



🧠 Features
📸 Scan Food Packages using image upload
🔍 OCR Extraction of nutrition & ingredients
🤖 AI Analysis (Groq) for health scoring
⭐ General & Personalized Ratings
⚠️ Health Warnings & Risk Tags
